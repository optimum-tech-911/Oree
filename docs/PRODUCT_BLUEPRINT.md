# Blueprint produit

## Positionnement

Orée est le centre de contrôle d'un projet de création de société : comprendre, choisir les structures à comparer, préparer les informations, centraliser les documents, échanger avec un conseiller et suivre les formalités.

La promesse centrale est la clarté et le suivi, pas un simple formulaire administratif.

## Modèle d'acquisition

Google Search est le moteur commercial initial. Chaque groupe d'intention reçoit une page dédiée :

- création SASU ;
- création EURL ;
- création SAS ;
- création SARL ;
- SASU ou EURL / choix du statut ;
- salarié qui prépare une transition ;
- micro-entrepreneur qui passe en société ;
- création à plusieurs ;
- dossier bloqué.

La page doit reprendre exactement le vocabulaire de la recherche et conduire au bon état du diagnostic.

## Parcours principal

```text
Annonce Google
→ page spécifique
→ diagnostic sans compte
→ orientation indicative
→ coordonnées / rendez-vous / sauvegarde
→ création du compte
→ espace projet
→ documents et échanges
→ contrôle humain
→ formalités
→ résultat commercial qualifié
```

## Conversion adaptative

- projet sous 30 jours : commencer le dossier ;
- comparaison : faire valider l'orientation ;
- salarié : sauvegarder la feuille de route ;
- activité existante : planifier la transition ;
- plusieurs fondateurs : inviter les associés ;
- blocage : faire vérifier le dossier.

## MVP livré

- expérience publique complète ;
- neuf entrées d'acquisition ;
- diagnostic adaptatif ;
- résultat ;
- réservation de démonstration ;
- connexion/inscription démo ou Supabase ;
- espace client complet ;
- espace opérations ;
- Guide Orée ;
- données réalistes ;
- architectures attribution, analytics et backend.
